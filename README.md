Jhesso's SimpleConverter:
Allows user to convert all kinds of units. Hopefully makes life a bit easier.

- goals:
	- support conversion for the most used units (and a few fun ones)
	- currency conversion(?)
	- start with an extensive console app
	- learn GUI developement and make a functioning GUI for the app
	- Possibly make it into an android/ios app
	- support linux, windows and osx operating systems (no windows only libraries etc.)
- written in C++
